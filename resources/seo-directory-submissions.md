## Free launch platforms

Launch-style platforms where a product is submitted once and ranked or featured. Columns show approximate authority, link type (dofollow / nofollow), submission queue or process, and what each platform tends to be best suited to.

| Platform | Authority | Link type | Queue | Best for |
| --- | --- | --- | --- | --- |
| [Product Hunt](https://www.producthunt.com) | Very High | Nofollow | Self-scheduled | Products with existing audience |
| [Hacker News](https://news.ycombinator.com/show) | Very High | Nofollow | None | Developer tools |
| [Reddit](https://www.reddit.com) | Very High | Nofollow | None | Any product (authentic posts) |
| [AlternativeTo](https://alternativeto.net) | Very High | Nofollow | Community | Alternatives to known tools |
| [Indie Hackers](https://www.indiehackers.com) | Very High | Dofollow | None | Bootstrapped SaaS |
| [SaaSHub](https://www.saashub.com) | Very High | Dofollow | None | Any SaaS |
| [BetaList](https://betalist.com) | Very High | Dofollow | 1–4 weeks | Pre-launch products |
| [Uneed](https://www.uneed.best) | High | Dofollow | Editorial | Design-forward products |
| [RankInPublic](https://rankinpublic.xyz) | Medium | Dofollow | None | Weekly competition |
| [DevHunt](https://devhunt.org) | Medium | Dofollow | GitHub PR | Developer tools |
| [OpenAlternative](https://openalternative.co) | Medium | Dofollow | Must be open source | Open-source projects |
| [Fazier](https://fazier.com) | Medium | Dofollow | Quick review | SaaS and AI tools |
| [MicroLaunch](https://microlaunch.net) | Medium | Dofollow | Quick review | Micro SaaS, MVP launches |
| [Startups.fyi](https://startups.fyi) | Medium | Dofollow | None | Additional backlink coverage |
| [SaaSCity](https://saascity.io) | Low | Dofollow | None | Gamified SaaS discovery |
